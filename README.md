# AuthWizard

**AuthWizard** is a simple, customizable, and secure authentication library designed to make integrating authentication into your applications fast and easy. Whether you're building a web app, mobile app, or microservice, AuthWizard handles user authentication, authorization, and session management.

## Features

- **OAuth 2.0 & OpenID Connect**: Support for major identity providers such as Google, GitHub, Facebook, and more.
- **JWT Token Support**: Secure, stateless authentication using JSON Web Tokens (JWT).
